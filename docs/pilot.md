# Pilot Program Design

## Overview

The SADP pilot program is designed to demonstrate the feasibility, effectiveness, and scalability of voucher-based ayuda disbursement in a controlled environment.

## Pilot Scope

### Coverage
- **Region:** One province or region (e.g., Region IVA - CALABARZON)
- **Beneficiaries:** 10,000 households
- **Voucher Value:** ₱50,000 each (adjustable based on program)
- **Duration:** 6 months (issuance → redemption → audit)

## Pilot Phases

| Phase | Key Activities | Duration |
|-------|----------------|----------|
| **1. Alignment** | Secure MOUs, technical sandbox setup | Month 1 |
| **2. Integration** | Link Government Bank APIs and x-Change ledger | Months 2–3 |
| **3. Issuance** | Release and distribute vouchers | Month 4 |
| **4. Redemption** | Monitor live redemptions and feedback | Months 5–6 |
| **5. Audit & Reporting** | Independent audit, publish results | Month 7 |

## Success Criteria

| Metric | Target | Critical? |
|--------|--------|-----------|
| Redemption rate | ≥85% | Yes |
| System uptime | ≥99.9% | Yes |
| Beneficiary satisfaction | ≥90% | Yes |
| Leakage reduction vs. baseline | ≥70% | Yes |
